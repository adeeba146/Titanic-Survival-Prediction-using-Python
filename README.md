# Titanic Survival Prediction using Python & Machine Learning

## Overview

This project analyzes the Titanic dataset and predicts passenger survival using machine learning techniques. The project includes data cleaning, exploratory data analysis, visualization, feature engineering, and predictive modeling.

---

## Problem Statement

The objective of this project is to analyze passenger data and identify the factors that influenced survival during the Titanic disaster.

---

## Objectives

* Perform data cleaning and preprocessing
* Analyze survival patterns
* Visualize important trends
* Build a machine learning prediction model
* Evaluate model performance

---

## Dataset Information

The dataset contains passenger details such as:

* Age
* Gender
* Passenger Class
* Fare
* Embarkation Port
* Survival Status

Dataset Source:
Titanic Dataset from Kaggle
